# Favicon 设计规范 - 快递哥布林 (Delivery Goblin)

## 设计概念

**主题**: The Delivery Goblin - 快递哥布林
- 代表玩家的角色
- 戴着兜帽或便帽的送货员剪影
- 露出非人类的发光眼睛（绿色/荧光色）
- 暗示主角是"快递哥布林"
- 黑色幽默和角色代入感

## 设计要求

### 视觉元素
1. **主体**: 戴兜帽/便帽的送货员剪影
2. **关键特征**: 
   - 发光眼睛（绿色 #22c55e 或荧光色）
   - 可能手持包裹或送货箱
   - 暗黑恐怖风格
3. **背景**: 
   - 深色背景 (#0a0e27 或黑色)
   - 可选：轻微的绿色光晕

### 尺寸要求 (根据 Google 规范)

必须提供的图标尺寸：

1. **favicon.ico** (16x16, 32x32, 48x48 - 多尺寸ICO文件)
2. **icon.svg** (可缩放矢量图)
3. **icon-16.png** (16x16)
4. **icon-32.png** (32x32)
5. **icon-96.png** (96x96)
6. **icon-192.png** (192x192) - PWA 使用
7. **icon-512.png** (512x512) - PWA 使用
8. **apple-icon.png** (180x180) - Apple Touch Icon
9. **icon-144.png** (144x144) - 可选，Windows Tile

### 设计要点

- **简洁**: 小尺寸下仍能清晰识别
- **对比**: 发光眼睛与深色背景形成强烈对比
- **识别度**: 即使缩到16x16也能看出是"戴帽子的生物"
- **品牌一致**: 与网站的暗黑恐怖主题一致

## 文件位置

所有图标文件应放在 `public/` 目录下：

```
public/
├── favicon.ico
├── icon.svg
├── icon-16.png
├── icon-32.png
├── icon-96.png
├── icon-192.png
├── icon-512.png
├── apple-icon.png (或 apple-touch-icon.png)
└── icon-144.png (可选)
```

## 设计草图描述

### 主视觉（SVG/高分辨率版本）
```
[深色背景]
    ╭─────╮
   ╱  ╱╲  ╲
  │   ╰─╯   │  <- 兜帽轮廓
  │  ◉  ◉   │  <- 发光眼睛
  │  ╱ ╲    │  <- 鼻子/嘴巴区域
  │  ╲ ╱    │
  │   ╳     │  <- 可选：包裹/箱子
   ╲       ╱
    ╰─────╯
```

### 关键设计元素：
- 眼睛: 绿色荧光 (#22c55e)，带光晕效果
- 兜帽: 深色剪影，边缘清晰
- 可选元素: 小包裹图标在底部
- 整体: 圆形或方形圆角，适合 favicon 显示
